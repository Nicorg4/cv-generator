# 🎨 CV Generator Frontend

This is the **frontend** for the CV Generator web app. It allows users to fill out a step-by-step form and generate a Curriculum Vitae in Harvard format.

## ⚙️ Tech Stack

- **React**
- **styled-components**
- **React Hooks**
- **Step-by-step wizard form**
- **Fetch API**
